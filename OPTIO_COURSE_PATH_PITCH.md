# Optio Course Path — Proposal

## Summary

Offer Williamsburg Learning students an optional alternate path for completing programming courses. Same curriculum, same Canvas enrollment, same end-of-semester grade — different way of getting there.

- **Traditional path** (default): students complete Skill Struck assignments; grades record in Canvas as today.
- **Optio path** (opt-in): students earn XP through self-directed and Skill Struck-based work, reviewed and approved by the teacher. Hitting the course XP target = A in Canvas.

Skill Struck remains fully available to Optio-path students as a learning resource — it just isn't the system of record for their grade.

## How it works for the student

1. **At enrollment**, the student (or parent) chooses a path: Traditional or Optio. Default is Traditional.
2. **Optio students** are enrolled in the corresponding Optio Course on the Optio platform. Each course is broken into 5–7 Projects (units), each with brief Lessons and a suggested Task menu.
3. **Tasks earn XP** when completed and approved. The Task menu includes both Skill Struck-based work (complete a module, submit evidence) and self-directed work (build a real project, teach a concept, write a tutorial, automate something they actually use). Students can also propose their own tasks.
4. **Submitted tasks enter "pending review."** The teacher approves the evidence, XP is awarded.
5. **Hitting the course XP target = course complete = A in Canvas.**
6. **No partial credit.** A student who hasn't hit the target by semester end receives WL's standard grade for an unfinished course — the Optio path doesn't change late-work policy.
7. **Path switching is allowed up to midterm.** After midterm, the choice is locked.

## How it works for Williamsburg

- **Canvas remains the system of record** for enrollment and final grade.
- **Skill Struck remains available** to all students; for Optio-path students it's a learning resource rather than a grading channel.
- **Grade output is binary**: A on completion, or whatever Williamsburg assigns for unfinished work. No new grade categories, no new transcript codes.
- **Transcript impact: none.** Final grades work exactly as they do today.

## The teacher's role

- Builds the Course structure (Projects, Lessons, suggested Tasks) once per course.
- Sets the XP target per course.
- Coaches students through choosing interesting, student-directed approaches that fit each Project's concepts — turning curriculum requirements into work the student actually wants to do.
- Reviews and approves student-submitted task evidence as the semester progresses, using Optio's existing review tooling.

## Worked example — Python 1

**Course XP target**: 1,000 XP (the standard for a semester-long Optio course at WL).

The Skill Struck Python 1 curriculum maps to 6 Optio Projects. Each Project covers the same concepts a Traditional-path student would learn through Skill Struck, with the matching Skill Struck mini-project serving as one anchor task option among several.

**Projects**:

1. **Hello, Python** — Intro, Variables, Input/Output, Syntax, Variable Types, Concatenation
   - Anchor task: Skill Struck "Mad Libs"
2. **Numbers & Math** — Numbers, Converting, Math, Modulus
   - Anchor task: Skill Struck "Simple Calculator"
3. **Strings** — Strings, String Methods, Checking Strings, Concatenating Numbers
   - Anchor tasks: Skill Struck "Reading Competition," "Decipher," "Story Maker"
4. **Lists** — Lists
   - Anchor task: Skill Struck "Team Roster"
5. **Decisions & Loops** — If Statements, Conditionals, Elif, For Loops
   - Anchor tasks: Skill Struck "Sentence Generator," "Earnings Calculator"
6. **Build Something Real** — student-defined capstone

**Sample task menu for "Hello, Python"**:

- Complete the Skill Struck "Mad Libs" project — submit code and a short reflection
- Build your own Mad Libs variant using a topic or story you actually like
- Make a profile generator that asks for name, age, and interests, then prints a styled bio
- Write a Python script that asks 3 questions and turns the answers into a fortune, weather report, or joke
- Teach a sibling or friend what a variable is — document or record the explanation

Students choose any combination of suggested tasks, propose their own, and use Skill Struck freely as a learning resource. Each task is worth XP toward the course target. The same content gets covered; the path through it is the student's.

## Why offer it

- **Engagement.** Students who learn best by building real things get a path that rewards that. Students who prefer structured assignments stay on the Traditional path. Both paths cover the same curriculum.
- **Portfolio.** Optio students leave the course with a portfolio of completed work — real artifacts, not just a grade.
- **Low risk.** The Traditional path is unchanged. Skill Struck stays in place. Canvas grading stays in place. The Optio path is purely additive.
