## How The Treehouse works on Optio

The Treehouse is a private space inside Optio that only Treehouse students and
facilitators can see (a "The Treehouse" tab in the menu). Students get a simple,
picture-first experience; facilitators get tools to notice, celebrate, and
document learning — built on top of Optio's existing quests, journal, and
school-jobs features.

---

## ✅ Built and ready to test

### For students (the kid experience)
- **Tap-your-photo login (kiosk).** On a shared tablet/Chromebook, a student taps
  their photo to sign in — no password. A "I'm done — let someone else use this"
  button hands the device to the next student.
- **Fullscreen mode.** On the kiosk the app goes fullscreen automatically — no
  menus or toolbars, just The Treehouse. A big **"← Back"** button is always on
  screen so a young student can get home with one tap.
- **Simple home screen.** A greeting, the student's coin balance, their most
  recent task to "keep going," and four big buttons: **My Quests**, **Find a
  Quest**, **Showcase**, **School Jobs**.
- **Browse quests by category.** Quests shown as colorful picture cards grouped by
  area (Creative, STEM, Wellness, Communication, Civics), with age labels.
- **"I Need Help" and "I'm Proud of This!"** buttons. Both alert the facilitator.
  After asking for help, the student sees friendly "while you wait, you could…"
  suggestions to keep their momentum.
- **One-tap task completion** without having to upload evidence first (the
  facilitator can add the photo/notes later) — designed for ages 5–9.
- **Showcase sign-up.** Students see the current showcase (theme, countdown,
  project ideas), sign up with a project name, and see "✅ You're signed up!" with
  an Edit button.

### For facilitators (Jennie's tools)
- **Facilitator dashboard** with tabs:
  - **Signals** — a live queue of who needs help and who's proud; mark each "Done."
  - **Pins** — completed quests ready to turn into physical pins/badges; batch
    "Mark created."
  - **Showcase** — create a showcase (title, theme, date, suggested project
    ideas), see each event's **roster** (who's presenting + project names), and
    **add students** to a roster yourself.
  - **Balances** — every student's coin balance with a quick +/- adjuster.
  - **Kiosk** — generate a device code to set up a shared tablet.
- **Notifications** when a student completes a task or quest, asks for help, is
  proud of something, or joins a showcase.
- **Age-appropriate task generation.** When the AI suggests tasks for a younger
  student, it now keeps them simple, hands-on, and quick — no more
  too-advanced wording for a 5–7 year old.
- **Cohorts.** Two groups exist ("Littles 5–7", "Bigs 8–13"). School Jobs can be
  limited to a chosen cohort so younger and older students see different jobs.
- **Existing Optio tools still apply:** quests/projects, the advisor dashboard
  (who needs attention), the journal, portfolios, and School Jobs (bounties).

## 🔜 Still to come (not built yet)

1. **AI help with documentation & portfolios** — the AI reading a facilitator's
   notes/photos and suggesting skill areas and growth traits (persistence,
   creativity, collaboration…), plus quarterly summaries, interest trends, and
   "highlight reels" for families. *(Biggest remaining piece.)*
2. **Tag many students in one observation** — post one photo + caption and have it
   land in several students' portfolios at once.
3. **Showcase project planning** — AI-suggested project steps with deadlines
   spaced backward from the showcase date, plus reminders and a timeline.
4. **Smaller items:** a one-screen quick "celebrate + document" capture for
   facilitators; a form to record a student's interests/goals; a cohort filter on
   the dashboard; an in-app way to set a quest's recommended age; audio
   read-aloud for early readers.
5. **Longer-term ideas** from the user stories (peer-collaboration quests, mentor
   roles, a marketplace for spending coins, QR-code pins, habit/feelings
   check-ins, ESA reporting) — noted for the future.

## How to test (once deployed)

**As a facilitator (Jennie):**
1. Log in with `thetreehouse.alc@gmail.com`.
2. Open the **The Treehouse** tab → **Facilitator Dashboard**.
3. Try each tab: Signals, Pins, Showcase (create one + view/add to roster),
   Balances, Kiosk (generate a device code).

**As a student (shared device):**
1. On the tablet, go to the kiosk page and enter the device code from the Kiosk tab.
2. Tap a student photo → you're in fullscreen on the student home.
3. Try the four buttons, "I Need Help"/"I'm Proud," complete a task, sign up for
   the showcase, then "I'm done" to hand off the device.
